# Freelance Platform AI Service

This is the AI/ML microservice for the Freelance Platform, hosted on Hugging Face Spaces using Docker.

## Features
- Resume Parsing
- Job Matching
- Salary Prediction
- Intent Classification for Chat

## Deployment Notes
This service is built with FastAPI and runs on port 7860.
